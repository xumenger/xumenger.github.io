## Agent介绍

比如我现在只有一个main agent，然后它有这些文件

![](../media/image/2026-03-22/01.png)

每个文件的作用如下：

|  文件        |  用途  |  加载时机 |
|  -----       | ------ |  ------- |
| AGENT.MD     |  操作指令、连续性规则、重要记忆 | 每会话首条消息
| SOUL.MD      |  人格定义、语气、边界 | 每会话首条消息
| TOOLS.MD     |  环境特定的工具说明 | 每会话首条消息
| USER.MD      |  用户信息、偏好、项目背景 | 每会话首条消息
| IDENTITY.MD  |  助手身份（名称、表情、头像） | 每会话首条消息
| MEMORY.MD    |  长期记忆（安全敏感内容） | 仅主会话

每个Agent 有自己的WorkSpace！

## clawhub Skills 市场

[http://clawhub.ai/](http://clawhub.ai/) 是OpenClaw 整合的各种Skills，执行下面的命令可以安装clawhub 的客户端工具

```shell
npm i -g clawhub
```

比如要安装一个联网搜索的Skill：[https://clawhub.ai/Moochmaniac/moochmaniac-tavily-search](https://clawhub.ai/Moochmaniac/moochmaniac-tavily-search)

可以执行下面的命令安装这个Skill

```shell
clawhub install "moochmaniac-tavily-search"
```

可以看到这个Skills 安装成功了

![](../media/image/2026-03-22/02.png)

但是如果你通过会话问OpenClaw 当前安装了哪些Skills，可能不会展示刚才安装的moochmaniac-tavily-search，那么需要重启一下OpenClaw Gateway，同时要新开一个聊天会话！

![](../media/image/2026-03-22/03.png)

```shell
openclaw gateway restart
```

## find skill 技能

这是一个可以帮忙搜索相关功能Skill 的技能，可以使用下面的命令完成安装（这种安装方式不是安装在WorkSpace 下面，而是安装在.openclaw 下面）

```shell
npx skills add https://github.com/vercel-labs/skills/tree/main/skills/find-skills
```

然后可以使用OpenClaw 帮忙找一些你需要的技能

![](../media/image/2026-03-22/04.png)

找到功能强大的Skill、自己实现功能强大的Skill，是玩转OpenClaw 的关键所在！
