## 搭建开发环境

创建D:\LLM\PyEnv\DeepLearn，作为虚拟文件夹

```shell
> cd D:\LLM\PyEnv\
> python3 -m venv DeepLearn
> .\DeepLearn\Scripts\activate
```

安装PyTorch 依赖

```shell
d:\llm\pyenv\DeepLearn\scripts\python.exe -m pip install --upgrade pip
pip3 install --upgrade setuptools
pip3 install pytorch
```
